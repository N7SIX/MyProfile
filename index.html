Key Contributions

📊 Spectrum Analyzer + Waterfall (Bandscope)

Extended the existing spectrum analyzer with a 4×4 Bayer 16-level grayscale dithering waterfall display, channel name overlay, and persistent settings storage on UV-K1 / UV-K5 V3 using the native flash path.

📶 IARU-Compliant S-Meter

Rewrote the S-Meter logic to comply with IARU Region 1 Technical Recommendation R.1 for VHF/UHF, replacing the Egzumer EEPROM-based level settings with hardcoded values. Two styles available: Classic and Tiny.

🛡️ Safety & Persistence Hardening (UV-K1/K5 V3)

Implemented low-voltage TX blocking at 7.0V, deferred save handling during unsafe battery conditions, versioned A/B settings snapshots with checksum validation, generation-based boot recovery, and write-readback verification retries — significantly reducing EEPROM/flash corruption risk.

🔑 Expanded Menu & UX Improvements

Added 15+ new menu entries (SetPwr, SetPTT, SetTOT, SetCtr, SetInv, SetEOT, SetMet, SetLck, SetGUI, TXLock, SetTmr, SetOff, SetNFM, TxLock, etc.), improved keyboard frequency input, new key shortcuts (F+↑/↓ squelch, F+F1/F2 step), and a redesigned status bar.

🌍 Regional Band Plans

Added frequency-lock band plans for HAM CA (Canadian zone), PMR 446, and FRS/GMRS/MURS to make the firmware compliant with regional amateur radio regulations out-of-the-box.

🐍 CMake + Docker Build System (UV-K1/K5 V3)

Migrated the UV-K1/K5 V3 firmware build to a modern CMake + Docker workflow with CMakePresets.json, making cross-platform reproducible builds accessible without installing a local toolchain.
